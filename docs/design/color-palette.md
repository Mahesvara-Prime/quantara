# Quantara — Color Palette Reference

Ce document regroupe les différentes palettes de couleurs étudiées pour la plateforme Quantara.

Contraintes de design :

- pas de blanc pur (#FFFFFF)
- pas de noir profond (#000000)
- peu de couleurs différentes
- confortable pour les yeux
- utilisable de jour comme de nuit
- style fintech / finance professionnel

## RÈGLES D’UTILISATION

Distribution recommandée des couleurs :

70% couleurs neutres (background / surfaces)
20% interface (texte / bordures)
10% accents (actions / graphiques)

Les couleurs de profit et perte doivent être utilisées
uniquement pour les graphiques ou les indicateurs de trading.
